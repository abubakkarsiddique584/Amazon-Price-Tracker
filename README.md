# 📦 Amazon Price Tracker

## 📌 Project Overview
This project is an **Amazon Price Tracker** built using Python. It automatically tracks the price of a product from Amazon and helps users monitor price changes over time. The goal is to help users decide the best time to buy a product when the price drops.

---

## 📁 Repository Contents
- `main.py` → Main Python script that tracks product price from Amazon  

---

## 🎯 Features
- Track Amazon product price in real-time 🛒  
- Extract product price using web scraping techniques  
- Compare current price with target price  
- Notify user when price drops (if implemented)  
- Simple and lightweight Python script  

---

## 🛠️ Technologies Used
- Python 🐍  
- Requests / BeautifulSoup (for web scraping)  
- Selenium (if used for dynamic pages)  
- HTML parsing  

---

## ⚙️ How It Works
1. The script takes an Amazon product URL  
2. It scrapes the product page  
3. Extracts the current price  
4. Compares it with your target price  
5. Alerts you if the price drops
